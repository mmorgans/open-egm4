# Exporting Data

The **Export Menu** (`e`) is a powerful, keyboard-driven interface designed for rapid data selection and export. It allows you to filter by specific plots and date ranges without leaving the keyboard.

## Workflow Overview

1. **Open Menu:** Press `e` from the Monitor screen.
2. **Filter Plots:** Press `p` to select which plots to include.
3. **Filter Dates:** Press `d` to select date ranges.
4. **Configure:** Press `f` (filename) or `l` (location).
5. **Export:** Press `e` to save the CSV.

## Date Menu (`d`)

The Date Menu offers multiple ways to filter data:

- **`c` (Calendar):** Opens a graphic 3-month calendar.
  - Use **Arrow Keys** to navigate.
  - **Space** toggles selection.
  - **Enter** confirms.
- **`r` (Range):** Type a range, e.g., `2024-01-01..2024-01-07`.
- **`b` (Before):** Type a date to select everything before it (e.g., `2024-06-01`).
- **`a` (After):** Type a date to select everything after it.

## Plot Menu (`p`)

- **Space**: Toggle selection of highlighted plot.
- **`a`**: Select **All**.
- **`n`**: Select **None**.

## Keyboard Shortcuts Summary

| Context | Key | Action |
| --- | --- | --- |
| Main Menu | `e` | **Export** now |
| | `p` | Open **Plots** filter |
| | `d` | Open **Dates** filter |
| | `f` | Set **Filename** |
| | `l` | Set **Location** |
| Date Menu | `c` | **Calendar** view |
| | `r` | Text **Range** input |
| | `b` / `a` | **Before** / **After** input |
| Any Filter | `q`, `Esc` | Cancel / Back |
| Any Filter | `Enter` | Confirm & Return |
