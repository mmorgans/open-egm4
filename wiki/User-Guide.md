# User Guide

## Monitor Screen

The Monitor Screen is your main dashboard. It displays real-time data from the EGM-4.

### Layout Overview
- **Left Panel:** Live Chart & Channel Legend.
- **Right Panel:** Statistics Table & Event Log.
- **Footer:** Available keyboard shortcuts.

### Keyboard Controls

| Key | Action |
| --- | --- |
| `1`-`9` | **Select Channel** to plot (e.g., `1`=CO2, `3`=PAR, `5`=Temp). |
| `=` / `-` | **Zoom In/Out** (Increase/Decrease time span). |
| `space` | **Pause/Resume** scrolling. |
| `c` | **Clear** chart history. |
| `e` | Open **Export Menu**. |
| `q` | **Quit** application. |

### Smart Features
- **Auto-Scaling:** Y-axis adjusts automatically to fit data, with smart padding.
- **Probe Detection:** Automatically switches between SRC-1 (Probe) and IRGA modes.
- **Zero Check:** Visual indicator when the device is performing a zero calibration.

---

## Connection Screen

On startup, Open-EGM4 attempts to auto-connect to known USB serial devices.
- **Countdown:** You have 5 seconds before it attempts connection.
- **Manual Select:** Press any key to cancel countdown and use `Up`/`Down` arrows + `Enter` to select a port manually.
